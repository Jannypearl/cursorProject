# Examples

## 示例 1：新增能力

**输入（改动摘要）**
- 新增迁移任务状态查询接口
- 增加任务 ID 参数校验
- README 补充调用示例

**输出（候选）**
1. 增加迁移任务状态查询接口并补充参数校验
2. 新增任务状态查询能力，完善 README 调用示例
3. 增加任务状态查询接口，修正入参校验缺失问题

## 示例 2：缺陷修复

**输入（改动摘要）**
- 修复源库类型被写死为 oracle 的问题
- 根据请求动态保存 sourceDbType

**输出（候选）**
1. 修复迁移任务源库类型写死为 oracle 的问题
2. 修复任务源库类型持久化错误，改为按请求动态保存
3. 修复 sourceDbType 写入逻辑，避免错误固定为 oracle

## 示例 3：优化改动

**输入（改动摘要）**
- 抽取重复的数据源创建逻辑到私有方法
- 补充异常日志上下文（任务 ID、数据源类型）

**输出（候选）**
1. 优化数据源创建逻辑并补充异常上下文日志
2. 重构重复的数据源组装代码，增强迁移失败可观测性
3. 优化数据源初始化流程，完善任务级异常日志信息
